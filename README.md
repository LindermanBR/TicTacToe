# Tic Tac Toe Game

Este é um simples jogo da velha implementado em Java. O jogo permite que dois jogadores joguem alternadamente até que um deles vença ou o tabuleiro esteja cheio.

## Funcionalidades

- Sorteio aleatório para determinar o jogador que inicia o jogo.
- Validação das jogadas para garantir que a posição escolhida pelo jogador seja válida.
- Verificação de vitória para determinar se um jogador ganhou o jogo.
- Verificação de empate para detectar quando o tabuleiro está cheio sem um vencedor.
- Exibição do tabuleiro após cada jogada para que os jogadores possam acompanhar o progresso do jogo.

## Estrutura do Projeto

O projeto possui a seguinte estrutura de arquivos:

- `App.java`: Classe principal do jogo que contém a lógica principal e a interação com o usuário.
- `README.md`: Este arquivo que fornece informações sobre o projeto.


## Executando o Jogo

Para executar o jogo, siga as etapas abaixo:

1. Certifique-se de ter o Java 8 (ou superior) instalado em seu sistema.
2. Clone ou faça o download do projeto para o seu ambiente de desenvolvimento.
3. Navegue até o diretório raiz do projeto.
4. Compile o código-fonte executando o comando `javac App.java`.
5. Execute o jogo com o comando `java App`.
